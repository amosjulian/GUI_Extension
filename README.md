# GUI_Extension
This project is a migrated and extended version of the original Simple GUI Extension (BlueJ v4) developed by the University of South Bohemia. The extension has been fully updated for BlueJ 5 compatibility, with additional features and improvements.

# Installation DE
Es wird vorausgesetzt, dass du BlueJ 5.X auf deinem System installiert hast.

0. Stelle sicher, dass BlueJ vollständig beendet ist.  
1. Lade die `GUI_Extension.jar` herunter.  
2. Öffne das BlueJ-Home-Verzeichnis. Dies sind die Standardordner von BlueJ:  
   - **Windows:** `C:\Program Files\BlueJ\`  
   - **macOS:** `/Applications/BlueJ.app/Contents/Resources/Java/`  
   - **Linux:** `/usr/share/bluej/`  
3. Kopiere die `.jar`-Datei nach `<BLUEJ_HOME>/lib/extensions2/` (unter macOS meistens `<BLUEJ_HOME>/extensions2/`).  
4. Starte BlueJ. Die Extension sollte automatisch geladen werden. Sie ist zu finden unter `Werkzeuge -> Simple GUI Extension`.  

# Installation EN
It is assumed that you have BlueJ 5.X installed on your system.

0. Make sure that BlueJ is completely closed.  
1. Download the `GUI_Extension.jar`.  
2. Open the BlueJ home directory. These are the default BlueJ folders:  
   - **Windows:** `C:\Program Files\BlueJ\`  
   - **macOS:** `/Applications/BlueJ.app/Contents/Resources/Java/`  
   - **Linux:** `/usr/share/bluej/lib/`  
3. Copy the `.jar` file to `<BLUEJ_HOME>/lib/extensions2/` (usually `<BLUEJ_HOME>/extensions2/` on macOS).  
4. Start BlueJ. The extension should load automatically. It can be found under `Tools -> Simple GUI Extension`.  

# Java GUI-Klasse speichern DE
Um in dieser Version von GUI-Extension die erstellte GUI in das BlueJ-Projekt einzubinden gehe auf `File-> Export Java Code`. Es wird empfohlen das GUI-Projekt vorher zu speichern (`File-> Save Project`).

⚠️ Wenn in dem Projekt bereits eine GUI_project-Klasse existiert wird diese beim Esport überschrieben!

# EN Export Java GUI-Class EN
To integrate the created GUI into the BlueJ project in this version of GUI-Extension, go to `File-> Export Java Code`. It is recommended to save the GUI project beforehand (`File-> Save Project`).

⚠️ If a GUI_project class already exists in the project, it will be overwritten during export!
